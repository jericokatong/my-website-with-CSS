Then I found the lesson about CRUD, that is a Program that Creating, Reading, Updating and Deleting. As beginner, I think it's a complex program because there is a saving process on external file. So even we have closed the program, data still there. In creating CRUD program, I use '.bin' external file and '.txt' external file, and there is a problem I found.
